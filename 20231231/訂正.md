#### Cmake実行時エラー

```
CMake Error at /usr/local/Cellar/cmake/3.28.1/share/cmake/Modules/Platform/iOS-Initialize.cmake:4 (message):
  iphoneos is not an iOS SDK
```

と出てきたら

```bash
sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer
```

と入力するとビルドが通るようになります。
